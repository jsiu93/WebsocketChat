# WebsocketChat
## FEATURE
1. 多人聊天（群聊）， 界面简洁美观， 使用ueditor支持发送文字，图片信息
2. 群聊成员列表， 登入登出公告
3. 存储聊天记录， 查看历史消息（按照日期筛选）
4. 一对一聊天
5. 并发处理
6. spring + redis
